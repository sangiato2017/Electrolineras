# Sistema de Gestión de Electrolineras - Bucaramanga

Este proyecto es un sistema basado en teoría de grafos e inteligencia artificial para simular, analizar y predecir rutas óptimas hacia puntos de carga (electrolineras) para vehículos eléctricos en el área metropolitana de Bucaramanga. 

## 📌 Requisitos Previos

* **Python 3.x** instalado en el sistema.
* Sistema Operativo: Linux (probado en Arch Linux), macOS o Windows.

## 🚀 Guía de Instalación y Configuración del Entorno

Debido a las políticas de seguridad en distribuciones modernas de Linux (como Arch Linux, que implementa el PEP 668 para evitar conflictos con el gestor de paquetes del sistema), es **obligatorio** ejecutar este proyecto dentro de un entorno virtual.

### Paso 1: Clonar o descargar el proyecto
Asegúrate de tener todos los archivos en una misma carpeta (por ejemplo, `Proyecto_Electrolineras`):
* `menuOpcionesE.py` (Script principal)
* `funcionesPreparacion.py` (Lógica y configuración de datos)

Abre tu terminal y navega hasta esa carpeta.
